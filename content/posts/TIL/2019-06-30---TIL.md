## 넘어야 할 장애물

- this와 function에 대한 개념 공부
  - 아래 코드에서 setCountFn의 this가 가리키는 것과 출력하는 것이 정확히 이해되지 않는다.

```javascript
App.ClickCounter = _data => {
  if (!_data) throw Error('_data')
  const data = _data
  data.value = data.value || 0

  return {
    getValue() {
      return data.value
    },

    count() {
      data.value++
    },

    setCountFn (fn) {
      this.count = () => (data.value = fn(data.value))
      return this
    }
  }
}
```
